# Fan-task-tic
## Tyler Ink Broeker
## Description
A simple application that gives the user the ability to write and save notes for each hour of a day using local storage. The current time, day, month and year are displayed at the top along with the userâ€™s time zone. Each hour is colored green when the current time is before it, red when it is the current hour, and grey when it is passed, the day resets at midnight.

## User-Story:
There are never enough hours in the day, and even when you have the time, managing in on your own can be troublesome. This simple application seeks to give a digital place for users to write down tasks or reminders for that day on the hour and save it by using local storage.
## Installation:
No installation, simple HTML/CSS/Javascript application
